# 🤖 AI 热门项目日报

每日自动更新 GitHub Trending 中的 AI/ML 热门项目

## 📋 功能特点

- **每日自动更新**：北京时间每天 9:00 自动抓取
- **AI 项目筛选**：自动识别和筛选 AI/ML 相关项目
- **趋势可视化**：展示近 14 天项目数量趋势
- **历史记录**：保留最近 30 天数据

## 🌐 访问地址

**https://interestsc1119.github.io**

## 🛠️ 技术架构

- **数据源**：GitHub Trending
- **自动化**：GitHub Actions
- **托管**：GitHub Pages
- **语言**：Python + HTML/CSS/JS

## 📁 项目结构

```
├── .github/workflows/  # GitHub Actions 配置
├── scripts/            # Python 脚本
├── data/               # JSON 数据文件
├── index.html         # 网页入口
└── README.md          # 说明文档
```

## 🚀 手动触发更新

1. 进入 Actions 页面
2. 选择 "Daily AI Trending Update"
3. 点击 "Run workflow"
